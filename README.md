## About probject
I have created ElectroNext e-commerce domain. It reloads items without loading page using react. i have created 17 items 
pages which includes all stuff from makeup, furniture, food, clothes, shows and kitchen things etc...and 
items can be added to cart and deleted too. Total price is shown and two themes are available dark and light. 
As per conditions i have used Dom, react router, redux toolkit, JS and React mainly , further i have deployed it too.
advanced feature i have added is theme toggle and real time data refresh and dashboard with charts.
